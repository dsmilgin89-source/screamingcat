## Build from Source

ScreamingCAT is open-source. You can build it from the source code.

**Prerequisites:** Node.js 18+, Rust 1.70+, system dependencies for Tauri 2

```
git clone https://github.com/[repo]/screamingcat.git
cd screamingcat
npm install
npm run tauri build
```

## FAQ

### Do I need to create an account?
No. Download, install, and start crawling. No registration required.

### Is this really free?
Yes. ScreamingCAT is open-source software released under a free license. There are no hidden costs, premium tiers, or feature gates.

### How do I update ScreamingCAT?
Download the latest version from this page and install it. Your saved project files (.sccat) are compatible across versions.

### Is my data sent anywhere?
No. ScreamingCAT runs entirely on your machine. Data is only sent externally when you explicitly use Google integrations with your own credentials.
